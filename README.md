## 📖 Sobre el Proyecto
EcoTrack es una plataforma web gamificada que busca que personas físicas y jurídicas puedan loguearse, seleccionar distintos planes y llevar un seguimiento de sus buenas prácticas de sustentabilidad. El objetivo principal es democratizar los buenos hábitos ambientales y hacer más disfrutable el reciclaje mediante un sistema de actividades semanales y recompensas.

## ✨ Funcionalidades Principales

### Para Usuarios (Personas y Empresas)
* **Registro y Planes:** Creación de perfiles eligiendo entre "Plan Gratuito" (Persona física) o "Plan Empresa" (Persona jurídica).
* **Cumplimiento de Tareas:** Participación en actividades semanales divididas en 5 ejes: Agua, Insumos, Energía, Residuos y Gestión Integral.
* **Subida de Evidencias:** Sistema para adjuntar archivos (JPG/PNG/PDF) comprobando la realización de las tareas.
* **Gamificación:** Barra de progreso en el dashboard para visualizar el puntaje acumulado en tiempo real.
* **Canje de Recompensas:** Intercambio de puntos por premios y alcance de nuevos hitos de nivel.
* **Eventos:** Inscripción a eventos presenciales o virtuales organizados por la comunidad.

### Para Administradores (Gerencia EcoTrack)
* **Gestión de Tareas:** Alta de nuevas actividades semanales con título, descripción, puntos otorgados y fecha de caducidad.
* **Validación:** Panel exclusivo para aprobar o rechazar las evidencias subidas por los usuarios.
* **Organización de Eventos:** Creación de eventos con fecha, cupo máximo y ubicación mediante un calendario interactivo.
