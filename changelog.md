# Changelog (Registro de Cambios)

Todos los cambios notables en el proyecto EcoTrack serán documentados en este archivo.

El formato está basado en [Keep a Changelog](https://keepachangelog.com/es-ES/1.0.0/), 
y este proyecto se adhiere a [Versionado Semántico](https://semver.org/lang/es/).

## [0.1.1] - 2026-04-12

### Add
- Maquetado menú

### Add
- **Base de Datos:** Creación del script SQL inicial.
- **Base de Datos:** Tablas para la gestión de tareas, divididas en 5 ejes (Agua, Insumos, Energía, Residuos y Gestión Integral).
- **Base de Datos:** Sistema relacional para el registro de hitos y canje de premios.
- **Documentación:** Archivo `README.md` con la información general y configuración inicial del proyecto.

## [0.1.0] - 2026-03-31
*Documentación inicial*

### Add
- Inicio de documento de Especificación de Requisitos (Versión 1.0).
- Definición del objetivo principal: plataforma para loguearse, seleccionar planes y seguir buenas prácticas de sustentabilidad.
- Definición de roles base: Gerente de EcoTrack (Admin) y Usuario final (Persona física/jurídica).
- Establecimiento de meta temporal: La primera versión del sistema tiene que estar disponible antes de Julio.
